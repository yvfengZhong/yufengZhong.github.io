STSM

STSM是基于深度图像分析的课堂教学效果自评系统（Self evaluation system of classroom teaching effect based on deep image analysis）的英文简称。STSM项目的目标，是通过人体骨骼点信息进行姿态分类，根据分类结果判断学生上课状态，从而对课堂教学效果作出评价。本项目只包含了“通过人体骨骼点信息进行姿态分类”部分代码，STSM项目的完整代码，有待补充。

本仓库包含以下内容：

  1. 一个README文件。
  2. 一个requirements.txt文件。
  3. 多个公开Kinect人体动作识别数据集。
  4. 一个适用于不同数据集的特征提取算法。
  5. 多个用于分类的机器学习算法。

安装

这个项目使用anaconda作为包管理器，使用pycharm作为IDE，基于 numpy1.19.4 和 sklearn0.23.2进行开发。请确保你本地安装了它们。更多依赖库，请查看requirements.txt文件。

使用说明

特征提取

本项目通过data_ker函数进行特征提取，现对data_ker函数原型进行简单介绍。

数据集空间特征提取

我们将数据集的每个动作文件，视作一个样本；将样本数据处理为（骨骼点数量，帧数量）大小的矩阵；计算矩阵的SPD矩阵作为样本的空间特征。

在这个过程中，有一个参数可由使用者调整：核函数。本项目包含机器学习中常用的一些核函数：线性核、协方差核、高斯核、拉普拉斯核、多项式核、sigmoid核。核函数的具体介绍可参考Kernels。

以HDM05数据集的处理文件data/dataset/hdm.py为例，介绍每个核函数的使用。

然而，选择如何选择核函数更好，或者如何选择核函数的参数更好，是一个依赖于经验的问题。可以单独运行hdm.py文件，设置第三个参数save==True，保存结果为txt文件，大概需要一分钟时间。观察txt文件，如果大量元素相同或趋紧于某一个数（如0，1），那么这个核函数的参数是不太好的；应该根据核函数的特点，调整参数，使元素均匀分布。当然更直接的办法，是在分类器中进行测试，如models/linear.py中进行测试，大概需要十多分钟左右的时间。

总而言之，先通过保存结果为txt文件，可以很快缩小参数的范围；然后对比linear.py文件的运行结果，可以得到最佳的参数。

数据集时间特征提取

本项目使用时间分片方法，提取骨骼点的时间特征。关于时间分片方法，可以参考Human Action Recognition Using a Temporal Hierarchy of Covariance Descriptors on 3D Joint Locations，这里简单介绍data_ker函数中包含的两个参数。
参数window表示时间分片的等级，当window=1时，时间分片等级为1，即只有一层，实际上不进行操作；当window=2时，时间分片等级为2，即有两层，第二层的矩阵行向量为第一层矩阵行向量的一半。参数overlap表示时间窗口的重叠关系，当overlap=True时，同一层的时间窗口，后一个时间窗口可以覆盖前一个时间窗口的一部分；当overlap=False时，同一层的时间窗口之间无交集。

模型训练

机器学习分类算法

本项目测试了sklearn中常见的分类算法，包括逻辑回归、支持向量机、K近邻、贝叶斯、决策树。对于每一个分类算法，使用网格搜索寻找最优参数。对于常见分类算法和网格搜索算法，在这里不做详细介绍，可以点击链接，查看sklearn文档。

集成算法

sklearn中常见的集成算法，包括voting、bagging、boosting。mlxtend包含了一个sklearn尚未实现的集成算法stacking。本项目最终选择了stacking算法，算法细节可以参考models/ensemble_stacking.py。
